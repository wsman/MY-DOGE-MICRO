# Amendment Archive

This directory stores supporting evidence for constitutional amendments.

`memory_bank/t0_core/amendment_log.md` is the current index. This archive keeps
the historical evidence that explains why each amendment was accepted.

Each record should include:

- Amendment identifier
- Date
- Changed principle or support document
- Reason for change
- Approver
- Before/after summary
- Related decision, review, or release evidence

Do not edit historical amendment evidence after approval. Add a superseding
amendment record instead.
