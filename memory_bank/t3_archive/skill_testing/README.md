# Skill Testing Archive

This T3 archive stores approved `/skill-test` and `/skill-improve` evidence for CDD's cross-project skill and agent testing system.

- `coverage-index.yaml` tracks latest test and improvement evidence.
- `results/` stores approved static, spec, category, and audit result reports.
- `improvements/` stores approved skill improvement records.

Do not place reusable test specs or rubrics here; those belong in the canonical
root `skill_testing/` directory. The memory-bank T2 mount is documented at
`memory_bank/t2_execution/skill_testing/README.md`.
