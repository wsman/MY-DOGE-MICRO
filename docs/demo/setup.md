# Research Copilot Demo Setup

## Demo Materials Status

The `demo_materials/` directory is currently a structure placeholder for the
interview workflow. The checked-in files are intentionally tiny stand-ins so
tests, local parsers, and route wiring have deterministic paths:

| File | Current status | Before interview |
|---|---|---|
| `market_summary_2026Q2.pdf` | text placeholder using a `.pdf` extension | Replace with a real annual report or market PDF |
| `sector_rotation.pptx` | text placeholder using a `.pptx` extension | Replace with a real earnings or sector-rotation deck |
| `financial_chart.png` | text placeholder using a `.png` extension | Replace with a real chart or terminal screenshot |
| `sample_portfolio.csv` | minimal demo CSV | Replace with a realistic, desensitized portfolio CSV if needed |

The current state should be read as: demo data structure is in place; final
multimodal content still needs operator-provided real materials.

## Tracking Policy

Keep placeholder files small and safe to commit. Do not commit proprietary,
client-confidential, or large binary interview materials. If real materials are
sensitive or heavy, keep them outside Git and copy them into `demo_materials/`
only on the demo machine.
