# OpenDoge Enterprise Research Copilot Demo

## Eight-minute script

0:00-1:00: Frame the enterprise research problem: long filings, slides, market data, and portfolio exposure need traceable analysis rather than a generic chatbot.

1:00-2:00: Open `Research Agent`, keep the default English question, and run the deterministic demo.

2:00-4:00: Walk through the timeline: model planning, `stock_overview`, `get_portfolio_exposure`, and approval request events.

4:00-5:30: Review the memo artifact and explain that material numbers are routed through tools.

5:30-6:30: Approve the high-risk publication action and show the memo artifact becoming available.

6:30-7:30: Explain the architecture split: Kimi plans and synthesizes; Python services validate; runtime controls state, trace, and approval.

7:30-8:00: Close with the production path: SSO, tenant ACL, PostgreSQL, Redis queue, object storage, gateway, observability, and SLA.
