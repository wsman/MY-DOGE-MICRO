# Kimi SA Demo Script

This page keeps interview/demo flow separate from product setup. It links the
existing demo material and evaluation guide instead of copying them.

## Demo Flow

1. Prepare local setup with [setup.md](setup.md).
2. Review sample prompts in [sample_prompts.md](sample_prompts.md).
3. Use the research copilot outline in
   [research_copilot_demo.md](research_copilot_demo.md).
4. Compare generated output with [sample_memo.md](sample_memo.md).
5. Run evaluation notes through [../guides/run-eval.md](../guides/run-eval.md).
